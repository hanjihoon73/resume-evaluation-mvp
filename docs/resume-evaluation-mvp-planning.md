- 화면 구성 및 정책
    - 홈 화면
        - 타이틀: “입사 지원자 자동 분석”
        - 서브타이틀: “이력서를 업로드하고, AI가 분석한 전문적인 리포트로 지원자를 평가하세요.”
        - 파일 업로드 및 분석 영역:
            - 파일을 선택해서 업로드하는 방식 (드래그 방식 불필요)
            - 2개 이상의 파일을 업로드해 한 번에 분석할 수 있음
            - 업로드 파일 이름의 형식: 포지션_이름_지원채널.pdf
            - [분석 시작] 버튼:
                - 파일 업로드 후 활성화
                - 활성화된 버튼을 클릭하면 분석 시작
            - 분석 진행 상황을 프로그래스바로 표시
            - 분석이 모두 완료되면 영역 초기화: 파일을 업로드하고 새로 분석을 할 수 있는 상태
        - AI 파일 분석 구현 시 필수 지침
            - 업로드된 파일의 분석 순서를 정한다.
            - 분석 순서는 지원채널, 포지션, 이름의 순서로 한글/알파벳 기준 오름차순으로 정한다.
            - 분석 순서 기준으로 n번 파일이 끝나면 n+1번 파일을 분석한다.
            - 파일 1개당 각 평가 항목별로 총 3회를 분석하고, 각 평가 항목별 평균 점수에 배점 비중을 반영해 총점을 계산한다.
            - 모든 분석 요청은 직렬로 처리한다. (병렬 처리 시 API 호출 과다)
        - 분석 결과 영역:
            - 분석 결과 목록의 정렬 순서: 지원채널, 포지션, 이름의 순서로 한글/알파벳 기준 오름차순
            - 분석 결과 목록 표시 정보:
                - 연번: 숫자
                - 이름: 업로드된 파일명에 포함된 지원자의 이름 (띄어쓰기 없음)
                - 포지션: 업로드된 파일명에 포함된 포지션 (띄어쓰기 없음)
                - 지원채널: 업로드된 파일명에 포함됨 지원채널 (띄어쓰기 없음)
                - 총점: AI 분석 결과로 계산된 총점
                - 적합도: 총점 기준의 적합도 등급 표시
                    - 90점 이상: 적극 추천
                    - 80점 이상 90점 미만: 추천
                    - 70점 이상 80점 미만: 조건부 추천
                    - 70점 미만: 부적합
                - [리포트 확인] 버튼: 버튼을 클릭하면 해당 지원자의 리포트 화면으로 이동
    - 리포트 화면
        - 타이틀: “지원자 분석 리포트”
        - 버튼:
            - [PDF 다운로드]:

                - PDF 파일 이름의 형식: 리포트_포지션_이름_지원채널.pdf
            - [목록으로 가기]: 버튼을 클릭하면 홈 화면으로 이동하고, 분석 결과 영역으로 자동 포커싱
        - 기본 정보 영역 표시 정보:

            - 포지션: 업로드된 파일명에 포함된 포지션 (띄어쓰기 없음)

            - 최종학력: 고졸/전문학사/학사/석사/박사 중 하나를 표시 / 정보를 찾을 수 없으면 ‘-’ 표시
            - 학점: 예) 3.5점 / 정보를 찾을 수 없으면 ‘-’ 표시
            - 나이: 예) 35세 / 생년이 있는 경우 ‘현재 연도 - 생년’의 결과 값을 나이로 표시하고, 정보를 찾을 수 없으면 ‘-’ 표시
            - 경력기간: 예) 5년 6개월 /  정보를 찾을 수 없거나 계산이 불가하면 ‘-’ 표시
        - 종합 평가 영역 표시 정보:
            - 총점: 예) 84점 / 100점
            - 적합도: 총점 기준의 적합도 등급 표시

        - 세부 평가 영역 표시 정보:

            - 각 평가 항목의 제목: 관련 전공/근속 기간/기술 스택/AI 역량/컬처핏

                - 텍스트 / 최소 200자, 최대 500자 이내
