# PDF 양식 최적화 계획 (v1.5 일관성 유지)

웹 리포트 화면의 UI 개편 사항을 PDF 문서에도 동일하게 반영하여 사용자에게 일관된 경험을 제공합니다.

## 수정 목표
- PDF에 지원자의 **전공(Major)** 정보를 추가합니다.
- 기본 정보 표시 순서를 웹과 동일하게 조정합니다. (포지션 > 이름 > 지원채널 > 전공 > 최종학력 > 학점 > 나이 > 경력기간)
- 불릿포인트 리스트 형식으로 바뀐 **총평**을 PDF에서도 리스트로 출력합니다.
- 웹의 새로운 레이아웃(점수/총평 분리, 상세분석 1열 배치)을 PDF 디자인에 최적화하여 반영합니다.

## 수정 파일
- [MODIFY] `src/components/ResumePDF.tsx`: 레이아웃 및 스타일 수정

## 확인 사항
- [react-pdf](https://react-pdf.org/) 라이브러리를 사용하므로 스타일링 제약 사항 내에서 웹의 느낌을 최대한 살립니다.
- 긴 총평 리스트가 페이지를 넘어갈 경우 적절히 처리되는지 확인합니다.
- 모든 필드가 데이터가 없을 때 '-' 또는 '정보 없음'으로 정상 표시되는지 확인합니다.
